# ✈️ Flight Finder App

A premium flight booking platform designed to dynamically highlight the cheapest available travel rates.

## Features
- Interactive Flight Search (Origin, Destination, Date)
- Automatic Cheapest Rate Sorting
- Login System & Passenger Checkout Flow
- Premium Admin Dashboard

---

## 💻 Developer Setup (With Node.js)

This project has been refactored into a full-fledged Single Page Application architecture. You must use Node.js to spin up the local development environments.

### Run Backend API
```bash
cd server
npm install
node app.js
```

### Run User Frontend (Vite)
Runs the main client application featuring flight search, login, and checkout mock flows.
```bash
cd client
npm install
npm run dev
```

### Run Admin Panel (Vite)
Runs the standalone admin interface.
```bash
cd admin
npm install
npm start
```
